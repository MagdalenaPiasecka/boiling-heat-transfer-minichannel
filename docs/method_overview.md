# Method Overview

This repository supports research on flow boiling heat transfer in annular and rectangular minichannels.

The project combines:
- experimental investigations,
- infrared thermography and flow visualization,
- CFD simulations,
- Trefftz-based inverse methods,
- machine-learning-assisted analysis.

The public repository is intended to provide reusable tools, example workflows, and selected public-ready data resources supporting reproducible research.
