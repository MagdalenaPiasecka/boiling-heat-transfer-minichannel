# Project Scope

## Title

Flow Boiling Heat Transfer in Annular and Rectangular Minichannels Using Machine Learning-Based Modeling

## Purpose of this public repository

This repository contains the public software and documentation layer of the research project focused on flow boiling heat transfer in annular and rectangular minichannels.

The repository is intended to support:
- reproducible data-processing workflows,
- public code for analysis and modelling,
- example notebooks,
- benchmark-ready example datasets,
- documentation of methods and assumptions.

## Scientific scope

The project investigates flow boiling and single-phase heat transfer in compact minichannel heat exchangers.

Key research dimensions:
- annular and rectangular minichannel geometries,
- controlled surface wettability,
- distilled water, dielectric fluids, and selected low-GWP refrigerants,
- high-speed imaging and infrared thermography,
- CFD simulations,
- Trefftz-based analytical-inverse modelling,
- machine-learning-based correlation development.

## Public repository boundaries

This repository may include:
- source code,
- notebooks,
- workflow documentation,
- selected example data,
- release-ready benchmark subsets,
- metadata and citation information.

This repository should not include:
- unpublished manuscript drafts,
- confidential administrative grant files,
- full raw experimental archives,
- internal meeting notes,
- private reviewer correspondence,
- institution-restricted material.

## Planned public outputs

Planned public outputs include:
- data-processing tools,
- model-development scripts,
- reusable analysis notebooks,
- selected validated datasets,
- release documentation,
- links to DOI-based archival records when available.

## Relation to the full project workspace

The full internal project workspace is maintained separately in a private repository.
The public repository is a curated subset intended for dissemination, reuse, and reproducibility.
