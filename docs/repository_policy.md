# Repository Policy

## Purpose

This is the public repository of the project.

## Intended content

This repository may contain:
- public code,
- workflow documentation,
- example notebooks,
- example datasets,
- selected figures,
- citation and metadata files.

## Excluded content

This repository should not contain:
- unpublished manuscript drafts,
- raw experimental archives,
- confidential project administration files,
- private reviewer correspondence,
- internal meeting notes,
- institution-restricted materials.

## Release rule

Only curated and explicitly approved materials should be added to this public repository.
